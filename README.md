# Fintech Market Prediction

## Flask App

### Market prediction 

run the flask app 
```
flask --app main.py run
```

enter `http://127.0.0.1:5000/` in your browser

### Market prediction + Real Time Stock Prices

run the flask app 

```
flask --app main_fusion.py run
```
